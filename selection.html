Selection Sort

Selection Sort adalah algoritma sorting yang bekerja dengan cara mencari nilai paling kecil, kemudian menempatkannya pada posisi yang sesuai.

Proses dilakukan berulang hingga seluruh data berada dalam keadaan terurut.

Konsep Selection Sort

Pada setiap iterasi, algoritma akan mencari data minimum dari array yang belum terurut.

Setelah nilai minimum ditemukan, data akan ditukar dengan posisi awal.

Karakteristik Selection Sort

  • Mencari nilai minimum
  • Menukar data sekali setiap iterasi
  • Cocok untuk data kecil
  • Algoritma sederhana

Algoritma Selection Sort 

1. Cari nilai minimum

2. Tukar dengan posisi awal

3. Geser indeks awal

4. Ulangi sampai data terurut

Program Selection Sort 

data = [7,2,5,10,1]

print("Data Awal =", data)

for i in range(len(data)-1):

    minimum = i

    for j in range(i+1, len(data)):

        if data[j] < data[minimum]:

            minimum = j

    data[i], data[minimum] = data[minimum], data[i]

print("Data Urut =", data)

Output Program 

Data Awal = [7, 2, 5, 10, 1]

Data Urut = [1, 2, 5, 7, 10]

Penjelasan Coding

Variabel minimum digunakan untuk menyimpan indeks nilai terkecil.

Perulangan kedua digunakan untuk mencari nilai minimum pada array.

Setelah nilai minimum ditemukan, data ditukar dengan posisi awal.

Study Kasus Selection Sort

Selection Sort dapat digunakan untuk mengurutkan berat barang dari terkecil ke terbesar.

Program Study Kasus 

berat = [10,2,5,8,1]

for i in range(len(berat)-1):

    minimum = i

    for j in range(i+1, len(berat)):

        if berat[j] < berat[minimum]:

            minimum = j

    berat[i], berat[minimum] = berat[minimum], berat[i]

print(berat)

Output Program 

[1, 2, 5, 8, 10]
