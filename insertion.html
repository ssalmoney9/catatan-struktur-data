Insertion Sort

Insertion Sort adalah algoritma sorting yang bekerja dengan cara menyisipkan data pada posisi yang sesuai.

Algoritma ini menganggap sebagian data di sebelah kiri sudah dalam keadaan terurut.

Konsep Insertion Sort

Data berikutnya akan dibandingkan dengan data yang sudah terurut, kemudian disisipkan pada posisi yang tepat.

Proses dilakukan berulang hingga seluruh data terurut.

Karakteristik Insertion Sort

  • Menggunakan konsep penyisipan data
  • Sebagian data dianggap sudah terurut
  • Cocok untuk data kecil
  • Stabil dalam sorting

Algoritma Insertion Sort 

1. Ambil data sebagai key

2. Bandingkan dengan data sebelumnya

3. Geser data lebih besar

4. Sisipkan key pada posisi tepat

5. Ulangi sampai seluruh data terurut

Program Insertion Sort 

data = [7,2,5,10,1]

print("Data Awal =", data)

for i in range(1, len(data)):

    key = data[i]

    j = i - 1

    while j >= 0 and key < data[j]:

        data[j+1] = data[j]

        j -= 1

    data[j+1] = key

print("Data Urut =", data)

Output Program 

Data Awal = [7, 2, 5, 10, 1]

Data Urut = [1, 2, 5, 7, 10]

Penjelasan Coding

Variabel key digunakan untuk menyimpan data yang akan disisipkan.

Perulangan while digunakan untuk membandingkan key dengan data sebelumnya.

Jika key lebih kecil, maka data digeser ke kanan.

Study Kasus Insertion Sort

Insertion Sort dapat digunakan untuk mengurutkan nilai ujian siswa dari terkecil ke terbesar.

Program Study Kasus 

nilai = [80,70,90,60,100]

for i in range(1, len(nilai)):

    key = nilai[i]

    j = i - 1

    while j >= 0 and key < nilai[j]:

        nilai[j+1] = nilai[j]

        j -= 1

    nilai[j+1] = key

print(nilai)

Output Program 

[60, 70, 80, 90, 100]
