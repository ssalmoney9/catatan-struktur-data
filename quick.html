Quick Sort

Quick Sort adalah algoritma sorting dengan konsep divide and conquer.

Algoritma ini bekerja menggunakan nilai pivot untuk membagi data menjadi dua bagian.

Konsep Quick Sort

Data yang lebih kecil dari pivot akan berada di sebelah kiri.

Data yang lebih besar dari pivot akan berada di sebelah kanan.

Algoritma Quick Sort 

1. Tentukan pivot

2. Partisi data

3. Data kecil ke kiri

4. Data besar ke kanan

5. Lakukan partisi kembali

6. Ulangi sampai data terurut

Program Partition 

def partition(A,p,q):

    x = A[p]

    i = p

    for j in range(p+1, q+1):

        if A[j] <= x:

            i = i + 1

            A[i], A[j] = A[j], A[i]

    A[p], A[i] = A[i], A[p]

    return i

a = [37,83,10,1,45,25,12,90,54]

print(
    partition(a,0,len(a)-1)
)

print(a)

Output Program 

4

[12,10,1,25,37,83,45,90,54]

Penjelasan Coding

Variabel x digunakan sebagai pivot.

Variabel i digunakan untuk indeks data kecil.

Jika data lebih kecil dari pivot, maka data ditukar.

Quick Sort Rekursif

Quick Sort menggunakan rekursif untuk melakukan partisi berulang.

Program Quick Sort 

def partition(A,p,q):

    x = A[p]

    i = p

    for j in range(p+1, q+1):

        if A[j] <= x:

            i = i + 1

            A[i], A[j] = A[j], A[i]

    A[p], A[i] = A[i], A[p]

    return i


def Quicksort(A,p,q):

    if p < q:

        r = partition(A,p,q)

        Quicksort(A,p,r-1)

        Quicksort(A,r+1,q)


a = [37,83,10,1,45,25,12,90,54]

Quicksort(a,0,len(a)-1)

print(a)

Output Program 

[1,10,12,25,37,45,54,83,90]
