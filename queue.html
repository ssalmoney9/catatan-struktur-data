Queue

Queue atau antrian adalah struktur data linear yang menggunakan konsep FIFO (First In First Out).

Data yang pertama masuk akan menjadi data pertama yang keluar.

Ilustrasi Queue

Konsep queue mirip seperti antrian di kasir. Orang yang datang lebih dulu akan dilayani lebih dulu.

Karakteristik Queue

  • Menggunakan konsep FIFO
  • Penambahan data dilakukan di REAR
  • Penghapusan data dilakukan di FRONT
  • Data pertama lebih dulu keluar

Operasi Pada Queue

  • Enqueue → menambahkan data
  • Dequeue → menghapus data depan
  • isEmpty → mengecek queue kosong
  • Size → menghitung jumlah data

Contoh Implementasi Queue Python 

def createQueue():
    q = []
    return q

def enqueue(q, data):
    q.insert(0, data)

def dequeue(q):
    data = q.pop()
    return data

def isEmpty(q):
    return q == []

def size(q):
    return len(q)

# Program utama

q = createQueue()

enqueue(q, 10)
enqueue(q, 20)
enqueue(q, 30)

print(q)

dequeue(q)

print(q)

Output Program 

[30, 20, 10]

[30, 20]

Penjelasan Coding

Fungsi enqueue digunakan untuk menambahkan data baru ke queue menggunakan insert().

Fungsi dequeue digunakan untuk menghapus data paling depan menggunakan pop().

Queue bekerja menggunakan konsep FIFO, sehingga data pertama akan keluar terlebih dahulu.

Study Kasus Queue

Queue sering digunakan pada sistem antrian, seperti antrian kasir, printer, dan rumah sakit.

Orang atau data yang datang lebih dulu akan diproses lebih dulu.

Contoh Study Kasus Antrian Kasir 

antrian = []

antrian.insert(0, "Andi")
antrian.insert(0, "Budi")
antrian.insert(0, "Citra")

print(antrian)

antrian.pop()

print(antrian)

Output Study Kasus 

['Citra', 'Budi', 'Andi']

['Citra', 'Budi']

Simulasi Permainan Ular Naga 

def createQueue():
    return []

def enqueue(q, data):
    q.insert(0, data)

def dequeue(q):
    return q.pop()

def size(q):
    return len(q)

def ularNaga(nama, hitungan):

    gameQueue = createQueue()

    for anak in nama:
        enqueue(gameQueue, anak)

    while size(gameQueue) > 1:

        for i in range(hitungan):
            enqueue(gameQueue, dequeue(gameQueue))

        dequeue(gameQueue)

    return dequeue(gameQueue)

pemenang = ularNaga(
    ['Andi','Rina','Sari','Dika'],
    2
)

print("Pemenang =", pemenang)

Kelebihan Queue

  • Cocok untuk sistem antrian
  • Data lebih teratur
  • Mudah diimplementasikan

Kekurangan Queue

  • Akses data tidak fleksibel
  • Hanya bisa menghapus dari depan
