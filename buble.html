Bubble Sort

Bubble Sort adalah algoritma sorting sederhana yang melakukan perbandingan dua data berdekatan, kemudian menukarnya jika urutannya salah.

Proses sorting dilakukan terus menerus sampai seluruh data berada dalam keadaan terurut.

Konsep Bubble Sort

Bubble Sort bekerja seperti gelembung. Nilai terbesar akan bergerak ke posisi paling akhir pada setiap iterasi.

Karakteristik Bubble Sort

  • Membandingkan data berdekatan
  • Menukar data jika salah urutan
  • Nilai terbesar bergerak ke kanan
  • Algoritma sederhana

Algoritma Bubble Sort 

1. Bandingkan dua data berdekatan

2. Tukar jika salah urutan

3. Geser ke kanan

4. Ulangi sampai seluruh data terurut

Program Bubble Sort 

data = [7,2,5,10,1]

print("Data Awal =", data)

for i in range(len(data)-1):

    for j in range(len(data)-1-i):

        if data[j] > data[j+1]:

            data[j], data[j+1] = data[j+1], data[j]

print("Data Urut =", data)

Output Program 

Data Awal = [7, 2, 5, 10, 1]

Data Urut = [1, 2, 5, 7, 10]

Penjelasan Coding

Perulangan pertama digunakan untuk menentukan jumlah iterasi.

Perulangan kedua digunakan untuk membandingkan data yang berdekatan.

Jika data kiri lebih besar dari data kanan, maka kedua data ditukar.

Study Kasus Bubble Sort

Bubble Sort dapat digunakan untuk mengurutkan nilai mahasiswa dari terkecil ke terbesar.

Program Study Kasus 

nilai = [80,70,90,60,100]

for i in range(len(nilai)-1):

    for j in range(len(nilai)-1-i):

        if nilai[j] > nilai[j+1]:

            nilai[j], nilai[j+1] = nilai[j+1], nilai[j]

print(nilai)

Output Program 

[60, 70, 80, 90, 100]
