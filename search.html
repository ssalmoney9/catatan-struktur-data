Sequential Search

Sequential Search adalah algoritma pencarian data dengan cara mengecek data satu per satu secara berurutan.

Pencarian dimulai dari indeks pertama hingga data ditemukan atau data habis.

Konsep Sequential Search

Sequential Search bekerja dengan membandingkan data yang dicari terhadap setiap elemen pada list.

Karakteristik Sequential Search

  • Pencarian dilakukan satu per satu
  • Tidak membutuhkan data terurut
  • Mudah dipahami
  • Cocok untuk data kecil

Jenis Sequential Search

Jenis Penjelasan
Unordered Sequential Search Digunakan untuk data acak
Ordered Sequential Search Digunakan untuk data terurut

Algoritma Sequential Search 

1. Mulai dari indeks pertama

2. Bandingkan data dengan key

3. Jika sama → data ditemukan

4. Jika tidak → lanjut ke data berikutnya

5. Berhenti saat data ditemukan
   atau data habis

Program Unordered Sequential Search 

def seqSearch(listData, data):

    ind = 0

    found = False

    while ind < len(listData) and not found:

        if listData[ind] == data:

            found = True

        else:

            ind = ind + 1

    return found

a = [12,5,9,8,1,10,26]

print(
    seqSearch(a,1)
)

Output Program 

True

Penjelasan Coding

Variabel ind digunakan untuk menyimpan indeks data.

Variabel found digunakan untuk mengecek apakah data ditemukan.

Perulangan while akan berjalan sampai data ditemukan.

Sequential Search Mengembalikan Index 

def seqSearch(listData, data):

    ind = 0

    found = False

    while ind < len(listData) and not found:

        if listData[ind] == data:

            found = True

        else:

            ind += 1

    if found:

        return ind

    else:

        return "Data tidak ada"

a = [12,5,9,8,1,10,26]

print(
    seqSearch(a,1)
)

Output Program 

4

Sequential Search Banyak Data Sama 

def seqSearchAll(listData, data):

    posisi = []

    for i in range(len(listData)):

        if listData[i] == data:

            posisi.append(i)

    if posisi:

        return posisi

    else:

        return "Data tidak ada"

a = [1,5,5,8,5,10]

print(
    seqSearchAll(a,5)
)

Output Program 

[1, 2, 4]

Ordered Sequential Search

Ordered Sequential Search digunakan untuk data yang sudah terurut.

Pencarian bisa berhenti lebih cepat jika data pada list lebih besar dari data yang dicari.

Ilustrasi Ordered Sequential Search 

DATA

[1, 5, 8, 10, 12, 26]


MENCARI 9


STEP 1

1 < 9


STEP 2

5 < 9


STEP 3

8 < 9


STEP 4

10 > 9


STOP

Data tidak ada

Program Ordered Sequential Search 

def orderedSeqSearch(listData, data):

    ind = 0

    found = False

    stop = False

    while (
        ind < len(listData)
        and not found
        and not stop
    ):

        if listData[ind] == data:

            found = True

        else:

            if listData[ind] > data:

                stop = True

            else:

                ind += 1

    if found:

        return ind

    else:

        return "Data tidak ada"

a = [1,5,8,10,12,26]

print(
    orderedSeqSearch(a,10)
)

Output Program 

3

Study Kasus Sequential Search

Sequential Search dapat digunakan untuk mencari data mahasiswa berdasarkan nama.

Program Study Kasus 

mahasiswa = [
    "Alya",
    "Budi",
    "Citra",
    "Dimas"
]

cari = "Citra"

found = False

for i in mahasiswa:

    if i == cari:

        found = True

if found:

    print("Data ditemukan")

else:

    print("Data tidak ditemukan")

Output Program 

Data ditemukan
