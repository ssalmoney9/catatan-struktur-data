Hashing

Hashing adalah teknik penyimpanan dan pencarian data menggunakan hash table dan hash function.

Dengan hashing, proses pencarian data dapat dilakukan lebih cepat.

Konsep Hashing

Data akan dipetakan ke slot tertentu menggunakan hash function.

Slot digunakan sebagai tempat penyimpanan data pada hash table.

Program Insert Data 

def remainderFunction(data, num):

    return data % num

def createHashTable(num):

    table = []

    for i in range(num):

        table.append('none')

    return table

def putData(data, table):

    for i in range(len(data)):

        ind = remainderFunction(
            data[i],
            len(table)
        )

        table[ind] = data[i]

    return table

a = [54,26,93,17,77,31]

hashTable = createHashTable(11)

print(
    putData(a, hashTable)
)

Output Program 

[77, 'none', 'none',
 'none', 26, 93,
 17, 'none', 'none',
 31, 54]

Program Search Hash 

def searchHash(data, table):

    hashVal = data % len(table)

    if data == table[hashVal]:

        return True

    else:

        return False

hashTable = [
    77,'none','none',
    'none',26,93,
    17,'none','none',
    31,54
]

print(
    searchHash(93, hashTable)
)

Output Program 

True

Collision

Collision terjadi ketika dua data memiliki nilai hash yang sama. 

77 % 11 = 0

44 % 11 = 0

Karena slot 0 sudah dipakai oleh 77, maka data 44 menyebabkan collision.

Penanganan Collision

Metode Penjelasan
Linear Probing Mencari slot kosong berikutnya
Quadratic Probing Melompat dengan pola kuadrat
Chaining Menyimpan banyak data dalam satu slot

Linear Probing

Linear probing mencari slot kosong satu per satu secara berurutan.

Program Linear Probing 

def linearProbing(ind, hashTable, data):

    count = ind

    found = False

    while (count != ind-1) and not found:

        if hashTable[count] == 'none':

            found = True

            hashTable[count] = data

        else:

            count = count + 1

            if count == len(hashTable)-1:

                count = 0

    return hashTable

Quadratic Probing

Quadratic probing mencari slot menggunakan pola kuadrat. 

h+1, h+4, h+9, h+16

Chaining

Chaining memungkinkan satu slot menyimpan lebih dari satu data.

Program Chaining 

table = [[] for i in range(11)]

def hash(x):

    return x % 11

def insert(table, key, value):

    index = hash(key)

    table[index].append(value)

insert(table,54,54)

insert(table,44,44)

insert(table,77,77)

print(table)

Output Program 

[[77,44],[],[],[],
 [],[],[],[],[],
 [],[54]]

Hashing String

String dapat diubah menjadi nilai integer menggunakan ASCII.

Program Hash String 

def strVal(strData):

    temp = 0

    for i in range(len(strData)):

        temp = temp + ord(strData[i])

    return temp

print(
    strVal('indonesia')
)

Output Program 

958
