Binary Search

Binary Search adalah algoritma pencarian data yang bekerja dengan membagi data menjadi dua bagian.

Binary Search hanya dapat digunakan pada data yang sudah terurut.

Konsep Binary Search

Pencarian dilakukan pada data bagian tengah terlebih dahulu.

Jika data lebih kecil dari nilai tengah, maka pencarian dilakukan ke kiri.

Jika data lebih besar, maka pencarian dilakukan ke kanan.

Karakteristik Binary Search

  • Data harus terurut
  • Membagi data menjadi dua bagian
  • Pencarian lebih cepat
  • Kompleksitas lebih kecil

Algoritma Binary Search 

1. Tentukan indeks tengah

2. Bandingkan data dengan middle

3. Jika sama → data ditemukan

4. Jika lebih kecil → cari kiri

5. Jika lebih besar → cari kanan

6. Ulangi sampai data ditemukan

Program Binary Search Iteratif 

def binarySearch(listData, data):

    first = 0

    last = len(listData) - 1

    found = False

    while first <= last and not found:

        midpoint = (first + last) // 2

        if listData[midpoint] == data:

            found = True

        else:

            if data < listData[midpoint]:

                last = midpoint - 1

            else:

                first = midpoint + 1

    return found

a = [4,6,10,34,56,78,99]

print(
    binarySearch(a,34)
)

Output Program 

True

Penjelasan Coding

Variabel midpoint digunakan untuk mencari indeks tengah.

Jika data lebih kecil dari midpoint, maka pencarian pindah ke kiri.

Jika data lebih besar, maka pencarian pindah ke kanan.

Binary Search Mengembalikan Index 

def binarySearch(listData, data):

    first = 0

    last = len(listData) - 1

    while first <= last:

        midpoint = (first + last) // 2

        if listData[midpoint] == data:

            return midpoint

        elif data < listData[midpoint]:

            last = midpoint - 1

        else:

            first = midpoint + 1

    return "Data tidak ada"

a = [4,6,10,34,56,78,99]

print(
    binarySearch(a,56)
)

Output Program 

4

Binary Search Rekursif 

def binarySearch2(listData, data):

    if len(listData) == 0:

        return False

    else:

        midpoint = len(listData) // 2

        if listData[midpoint] == data:

            return True

        else:

            if data < listData[midpoint]:

                return binarySearch2(
                    listData[:midpoint],
                    data
                )

            else:

                return binarySearch2(
                    listData[midpoint+1:],
                    data
                )

a = [4,6,10,34,56,78,99]

print(
    binarySearch2(a,78)
)

Output Program 

True

Study Kasus Binary Search

Binary Search digunakan untuk mencari data nilai siswa pada list yang sudah terurut.

Program Study Kasus 

nilai = [60,70,80,90,100]

cari = 90

first = 0

last = len(nilai)-1

found = False

while first <= last and not found:

    middle = (first + last) // 2

    if nilai[middle] == cari:

        found = True

    else:

        if cari < nilai[middle]:

            last = middle - 1

        else:

            first = middle + 1

print(found)
