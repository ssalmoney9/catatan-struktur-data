Shell Sort

Shell Sort adalah algoritma sorting yang merupakan pengembangan dari Insertion Sort.

Shell Sort bekerja dengan membandingkan data berdasarkan jarak tertentu yang disebut gap.

Konsep Shell Sort

Pada awal proses, data dibandingkan menggunakan gap besar.

Gap akan terus diperkecil sampai menjadi 1.

Karakteristik Shell Sort

  • Pengembangan insertion sort
  • Menggunakan gap
  • Sorting lebih cepat
  • Cocok untuk data sedang

Algoritma Shell Sort 

1. Tentukan gap

2. Bandingkan data berdasarkan gap

3. Tukar jika salah urutan

4. Kecilkan gap

5. Ulangi sampai gap = 1

Program Shell Sort 

def shellSort(data):

    gap = len(data) // 2

    while gap > 0:

        for i in range(gap, len(data)):

            temp = data[i]

            j = i

            while (
                j >= gap
                and data[j-gap] > temp
            ):

                data[j] = data[j-gap]

                j = j-gap

            data[j] = temp

        gap = gap // 2

data = [35,33,42,10,14,19,27,44]

shellSort(data)

print(data)

Output Program 

[10,14,19,27,33,35,42,44]

Penjelasan Coding

Variabel gap digunakan untuk menentukan jarak perbandingan data.

Gap awal bernilai setengah dari jumlah data.

Setelah satu proses selesai, gap akan diperkecil.

Insertion Sort vs Shell Sort

Insertion Sort Shell Sort
Tidak menggunakan gap Menggunakan gap
Lebih lambat Lebih cepat
Cocok data kecil Cocok data sedang

Study Kasus Shell Sort

Shell Sort dapat digunakan untuk mengurutkan nilai siswa dengan lebih cepat dibanding insertion sort biasa.

Program Study Kasus 

nilai = [88,75,90,60,95]

shellSort(nilai)

print(nilai)

Output Program 

[60,75,88,90,95]

Kelebihan dan Kekurangan

Kelebihan Kekurangan
Lebih cepat dari insertion sort Tidak stabil
Mudah diimplementasikan Pemilihan gap mempengaruhi performa
Efisien untuk data sedang Masih kalah dari quick sort
