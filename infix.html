Infix, Prefix, dan Postfix

Infix, Prefix, dan Postfix adalah bentuk penulisan ekspresi matematika yang digunakan pada struktur data stack.

Infix

Infix adalah bentuk penulisan operator di tengah operand. 

A + B

A * B

(A + B) * C

Prefix

Prefix adalah bentuk penulisan operator di depan operand. 

+ A B

* A B

* + A B C

Postfix

Postfix adalah bentuk penulisan operator di belakang operand. 

A B +

A B *

A B + C *

Perbedaan Bentuk 

INFIX

A + B


PREFIX

+ A B


POSTFIX

A B +

Prioritas Operator

  • () memiliki prioritas tertinggi
  • Pangkat lebih tinggi dari perkalian
  • Perkalian lebih tinggi dari penjumlahan

Konversi Infix ke Postfix

Konversi infix ke postfix menggunakan stack. 

INFIX

(A + B) * C


STEP 1

A B +


STEP 2

A B + C *


HASIL POSTFIX

AB+C*

Konversi Infix ke Prefix

Langkah konversi:

  • Balik ekspresi
  • Tukar tanda kurung
  • Ubah menjadi postfix
  • Balik hasil postfix 
INFIX

(A + B) * C


REVERSE

C * ( B + A )


POSTFIX

CBA+*


HASIL PREFIX

*+ABC

Program Infix ke Postfix 

def infixToPostfix(infix):

    stack = []

    output = []

    prioritas = {
        '+':1,
        '-':1,
        '*':2,
        '/':2
    }

    for char in infix:

        if char.isalpha():

            output.append(char)

        elif char == '(':

            stack.append(char)

        elif char == ')':

            while stack and stack[-1] != '(':

                output.append(stack.pop())

            stack.pop()

        else:

            while (
                stack and
                stack[-1] != '(' and
                prioritas[char] <= prioritas[stack[-1]]
            ):

                output.append(stack.pop())

            stack.append(char)

    while stack:

        output.append(stack.pop())

    return ''.join(output)

print(
    infixToPostfix("(A+B)*C")
)

Output Program 

AB+C*

Program Infix ke Prefix 

def infixToPrefix(infix):

    infix = infix[::-1]

    hasil = ""

    for i in infix:

        if i == '(':
            hasil += ')'

        elif i == ')':
            hasil += '('

        else:
            hasil += i

    stack = []

    output = []

    prioritas = {
        '+':1,
        '-':1,
        '*':2,
        '/':2
    }

    for char in hasil:

        if char.isalpha():

            output.append(char)

        elif char == '(':

            stack.append(char)

        elif char == ')':

            while stack and stack[-1] != '(':

                output.append(stack.pop())

            stack.pop()

        else:

            while (
                stack and
                stack[-1] != '(' and
                prioritas[char] <= prioritas[stack[-1]]
            ):

                output.append(stack.pop())

            stack.append(char)

    while stack:

        output.append(stack.pop())

    prefix = ''.join(output[::-1])

    return prefix

print(
    infixToPrefix("(A+B)*C")
)

Output Program 

*+ABC

Program Evaluasi Postfix 

def postfixEval(expr):

    stack = []

    token = expr.split()

    for i in token:

        if i.isdigit():

            stack.append(int(i))

        else:

            op2 = stack.pop()

            op1 = stack.pop()

            if i == '+':
                hasil = op1 + op2

            elif i == '-':
                hasil = op1 - op2

            elif i == '*':
                hasil = op1 * op2

            elif i == '/':
                hasil = op1 / op2

            stack.append(hasil)

    return stack.pop()

print(
    postfixEval("4 5 6 * -")
)

Output Program 

-26

def prefixEval(expr):

    stack = []

    token = expr.split()[::-1]

    for i in token:

        if i.isdigit():

            stack.append(int(i))

        else:

            op1 = stack.pop()

            op2 = stack.pop()

            if i == '+':
                hasil = op1 + op2

            elif i == '-':
                hasil = op1 - op2

            elif i == '*':
                hasil = op1 * op2

            elif i == '/':
                hasil = op1 / op2

            stack.append(hasil)

    return stack.pop()

print(
    prefixEval("- 4 * 5 6")
)

Output Program 

-26
