Deque

Deque atau Double Ended Queue adalah struktur data yang memungkinkan penambahan dan penghapusan data dari dua sisi.

Berbeda dengan queue biasa, deque dapat melakukan operasi di bagian front maupun rear.

Ilustrasi Deque

Deque mirip seperti antrean yang bisa masuk dan keluar dari depan maupun belakang.

Karakteristik Deque

  • Memiliki dua ujung yaitu front dan rear
  • Data bisa ditambah dari depan dan belakang
  • Data bisa dihapus dari depan dan belakang
  • Gabungan konsep stack dan queue

Operasi Pada Deque

  • addFront → menambah data depan
  • addRear → menambah data belakang
  • removeFront → menghapus data depan
  • removeRear → menghapus data belakang
  • isEmpty → mengecek deque kosong
  • size → menghitung jumlah data

Contoh Implementasi Deque Python 

def createDeque():
    d = []
    return d

def addFront(d, data):
    d.insert(0, data)

def addRear(d, data):
    d.append(data)

def removeFront(d):
    return d.pop(0)

def removeRear(d):
    return d.pop()

def isEmpty(d):
    return d == []

def size(d):
    return len(d)

# Program utama

deq = createDeque()

addFront(deq, "Struktur")
addRear(deq, "Data")
addFront(deq, "Python")

print(deq)

removeRear(deq)

print(deq)

Output Program 

['Python', 'Struktur', 'Data']

['Python', 'Struktur']

Penjelasan Coding

Fungsi addFront digunakan untuk menambahkan data di bagian depan deque menggunakan insert(0,data).

Fungsi addRear digunakan untuk menambahkan data di bagian belakang menggunakan append().

removeFront menghapus data paling depan, sedangkan removeRear menghapus data paling belakang.

Perbedaan Stack, Queue, dan Deque

  • Stack → akses satu sisi
  • Queue → masuk belakang keluar depan
  • Deque → akses dua sisi

Study Kasus Deque

Deque sering digunakan pada aplikasi browser, playlist musik, dan sistem undo-redo.

Salah satu implementasi deque adalah pengecekan kata palindrom.

Pengecekan Palindrom

Palindrom adalah kata yang tetap sama jika dibaca dari depan maupun belakang.

Contoh:

  • KATAK
  • KASUR RUSAK
  • MALAM

Contoh Program Palindrom 

def createDeque():
    return []

def addRear(d, data):
    d.append(data)

def removeFront(d):
    return d.pop(0)

def removeRear(d):
    return d.pop()

def size(d):
    return len(d)

def cekPalindrom(kata):

    d = createDeque()

    for huruf in kata:
        addRear(d, huruf)

    cek = True

    while size(d) > 1:

        depan = removeFront(d)
        belakang = removeRear(d)

        if depan != belakang:
            cek = False

    return cek

print(cekPalindrom("katak"))
print(cekPalindrom("mobil"))

Output Program 

True

False

Kelebihan Deque

  • Fleksibel karena dua arah
  • Cocok untuk data dinamis
  • Operasi lebih cepat

Kekurangan Deque

  • Implementasi lebih kompleks
  • Penggunaan memori lebih besar
