Rekursif

Rekursif adalah teknik pemrograman yang memanggil fungsi itu sendiri secara berulang.

Teknik ini digunakan untuk memecahkan masalah besar menjadi masalah yang lebih kecil.

Konsep Rekursif

Fungsi rekursif akan terus memanggil dirinya sendiri sampai mencapai kondisi berhenti.

Kondisi berhenti disebut base case.

Visualisasi Rekursif

Visualisasi rekursif dapat dilihat melalui visualizer berikut.

Buka Visualisasi Rekursif

Struktur Fungsi Rekursif 

def fungsi():

    if kondisi:
        return

    else:
        fungsi()

Aturan Rekursif

  • Harus memiliki base case
  • Harus memanggil dirinya sendiri
  • Masalah harus semakin kecil

Rekursif dan Iteratif

Rekursif menggunakan pemanggilan fungsi, sedangkan iteratif menggunakan perulangan.

Rekursif Iteratif
Menggunakan fungsi Menggunakan loop
Lebih sederhana Lebih cepat
Menggunakan stack memory Memori lebih kecil

Faktorial Rekursif

Faktorial adalah perkalian bilangan dari n sampai 1. 

5! = 5 × 4 × 3 × 2 × 1

Visualisasi Faktorial 

factorial(5)

5 × factorial(4)

5 × 4 × factorial(3)

5 × 4 × 3 × factorial(2)

5 × 4 × 3 × 2 × factorial(1)

5 × 4 × 3 × 2 × 1

Program Faktorial Rekursif 

def factorial(n):

    if n <= 1:

        return 1

    else:

        return n * factorial(n-1)

print(
    factorial(5)
)

Output Program 

120

Penjelasan Coding

Jika nilai n kurang dari atau sama dengan 1, maka fungsi berhenti.

Jika tidak, fungsi akan memanggil dirinya sendiri.

Triangular Numbers

Triangular Numbers adalah deret: 

1, 3, 6, 10, 15, 21

Visualisasi Triangular Numbers 

triangle(4)

4 + triangle(3)

4 + 3 + triangle(2)

4 + 3 + 2 + triangle(1)

4 + 3 + 2 + 1

= 10

Program Triangular Numbers 

def triangular(n):

    if n == 1:

        return 1

    else:

        return n + triangular(n-1)

print(
    triangular(4)
)

Output Program 

10

Towers of Hanoi

Towers of Hanoi adalah permainan memindahkan piringan dari satu tiang ke tiang lain.

Visualisasi Towers of Hanoi

Visualisasi Towers of Hanoi dapat dilihat melalui website berikut.

Buka Visualisasi Hanoi

Program Towers of Hanoi 

def hanoi(n, asal, bantuan, tujuan):

    if n == 1:

        print(
            "Pindah piringan 1 dari",
            asal,
            "ke",
            tujuan
        )

    else:

        hanoi(
            n-1,
            asal,
            tujuan,
            bantuan
        )

        print(
            "Pindah piringan",
            n,
            "dari",
            asal,
            "ke",
            tujuan
        )

        hanoi(
            n-1,
            bantuan,
            asal,
            tujuan
        )

hanoi(3,'A','B','C')

Output Program 

Pindah piringan 1 dari A ke C

Pindah piringan 2 dari A ke B

Pindah piringan 1 dari C ke B

Pindah piringan 3 dari A ke C
