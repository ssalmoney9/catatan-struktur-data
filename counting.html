Counting Sort

Counting Sort adalah algoritma sorting yang bekerja dengan menghitung jumlah kemunculan setiap data.

Setelah jumlah data diketahui, data akan ditempatkan pada posisi yang sesuai.

Konsep Counting Sort

Counting Sort tidak membandingkan data secara langsung seperti bubble sort atau quick sort.

Algoritma ini menghitung frekuensi setiap data.

Karakteristik Counting Sort

  • Menghitung jumlah data
  • Tidak menggunakan perbandingan
  • Sangat cepat
  • Cocok untuk integer kecil

Algoritma Counting Sort 

1. Hitung jumlah setiap data

2. Hitung jumlah kumulatif

3. Tempatkan data pada posisi tepat

4. Hasil data menjadi terurut

Konsep Kumulatif

Nilai kumulatif digunakan untuk menentukan posisi akhir suatu data pada array hasil. 

COUNT

[0,3,2,1,2]


KUMULATIF

[0,3,5,6,8]

Program Counting Sort 

def countingSort(A,k):

    C = []

    B = []

    for i in range(k+1):

        C.append(0)

    for j in range(len(A)):

        C[A[j]] = C[A[j]] + 1

        B.append(0)

    for i in range(1,k+1):

        C[i] = C[i] + C[i-1]

    for j in range(len(A)-1,-1,-1):

        B[C[A[j]]-1] = A[j]

        C[A[j]] = C[A[j]] - 1

    return(B)

a_list = [
    54,26,93,17,
    1,31,2,55,
    20,93,20,
    3,4,5
]

print(
    countingSort(a_list,93)
)

Output Program 

[1,2,3,4,5,17,20,20,
26,31,54,55,93,93]

Penjelasan Coding

Array C digunakan untuk menyimpan jumlah kemunculan data.

Array B digunakan untuk menyimpan hasil sorting.

Loop pertama menghitung jumlah data.

Loop kedua menghitung jumlah kumulatif.

Loop terakhir menempatkan data pada posisi yang tepat.

Quick Sort vs Counting Sort

Quick Sort Counting Sort
Menggunakan perbandingan Menggunakan perhitungan
Cocok untuk semua data Cocok integer kecil
Lebih fleksibel Lebih cepat

Kelebihan dan Kekurangan

Kelebihan Kekurangan
Sangat cepat Membutuhkan memori besar
Kompleksitas kecil Hanya untuk integer
Tidak menggunakan comparison Tidak cocok data besar acak

Kompleksitas Counting Sort

Kasus Kompleksitas
Best Case O(n+k)
Average Case O(n+k)
Worst Case O(n+k)
